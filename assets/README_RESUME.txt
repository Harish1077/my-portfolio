HOW TO USE YOUR OWN RESUME (DOWNLOAD BUTTON)

The portfolio "Download Resume" buttons use this file:

  Harish_Ragavendra_Resume.pdf

Steps:
1. Export your resume as a PDF (from Word, Google Docs, Canva, etc.).
2. Copy it into this folder:  PROTFOLIO\assets\
3. Replace the existing file by naming yours exactly:

     Harish_Ragavendra_Resume.pdf

4. Refresh the website and click "Download Resume" again.

If you prefer a different filename, change both links in index.html
(search for "Harish_Ragavendra_Resume.pdf").
